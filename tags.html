重要なHTMLタグ一覧

このページは、Railsチュートリアル 「HTML編」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページと して作られます。このページの制作を通じて、HTMLのテーブル（table）をtable タグやそれに関連するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLについてはここで参照できます。

ドキュメントタグ（document tags）

HTMLタグ タグ名 目的
<html> HTML文書（HTML document） HTML文書全体を囲むためのタグ
<head> ヘッド（head） メタ情報、スタイル、スクリプトなど、本文以外の情報を配置する場所
<body> ボディ（body） ウェブページのコンテンツ全体を囲むためのタグ
<title> タイトル（title） ブラウザのタブやウィンドウのタイトルバーに表示される文書のタイトルを指定
<meta> メタ情報（meta information） 文書の文字セット、ビューポートの設定、著者情報などのメタ情報を指定

ブロック要素（block element）

HTMLタグ タグ名 目的
h1–h6 見出し（heading） この中に見出しを置く（レベル: 1–6）
p 段落（パラグラフ: paragraph） この中に文章を入れる
table 表（テーブル: table） この中にテーブルを置く
tr 表の行（table raw） この中に行データを置く
th 表の見出し（table header） この中に表の見出しを置く
td 表のデータ（table data） この中に表のセルデータを置く
header ヘッダー（header） この中にウェブページのヘッダーを置く
div ページの区切り（division） この中にコンテンツのまとまりを置く
ol 順序付きリスト（ordered list） この中に順序付きのリストアイテムを置く
ul 順序なしリスト（unordered list） この中に順序なしのリストアイテムを置く
li リストアイテム（list item） <il>または<ul>内で項目を表示する
blockquote 長い引用（blockquote） 他のサイトやソースからの長い引用を表現する
header ヘッダー（header） ウェブページのヘッダー部分を置く
style スタイル（style） ページ内のCSSスタイルを定義する
link 外部リンク（link） 外部のCSSファイルやアイコンをリンクする

インライン要素（inline element）

HTMLタグ タグ名 目的 記述例 画面上の表示結果
em 軽い強調（emphasized） テキストを軽く強調する 技術の成熟（<em>technical sophistication</em>） 技術の成熟（technical sophistication）
strong 強調（strong） テキストを強調する <strong>少なくとも10億人</strong> 少なくとも10億人
a アンカー（anchor） ハイパーリンクにする <a href="https://learnenough.com/">Learn Enough</a> Learn Enough
img 画像（image） 画像を配置する <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code コード（code） 等幅フォントで表示する（monospace font） 関数は <code>def foo()</code> のように 関数は def foo() のように
span スパン（span） 特定のテキスト部分をグループ化またはスタイルを適用する 文章中の <span>特定の部分</span> を強調する 文章中の特定の部分を強調する
